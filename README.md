# TextureGen Pro

A high-fidelity PBR texture generation tool for Windows. Converts single images into full PBR sets (Albedo, Normal, Roughness, AO, Displacement) using algorithmic frequency separation.

## Features
- **Professional Quality:** Uses "Delighting" to remove baked shadows from Albedo.
- **Dual-Frequency Normals:** Blends micro-details and macro-structure for sculpted look.
- **Windows App:** Modern Dark Mode GUI.
- **Unreal Engine Ready:** Includes an importer script for 1-click UE5 integration.

## Installation (Windows)

1. **Download** this repository.
2. Double-click `build_app.bat` to generate the `.exe`.
3. The app will appear in the `dist/` folder.

## Usage

1. Open `TextureGenPro.exe`.
2. Load an image.
3. Adjust sliders (Normal Strength, Roughness, etc.).
4. Click **EXPORT ALL MAPS**.

## Unreal Engine Integration

1. Copy `scripts/unreal_importer.py` to your Unreal Project's python folder.
2. Run it to automatically import materials generated by this tool.
